Đọc và ghi File trong Java

   

Java I/O

   

Thao tác với file (Input/Output - I/O) là một kỹ năng thiết yếu trong lập trình Java. Các lớp phổ biến và hiệu quả nhất để xử lý file văn bản là sử dụng bộ đệm (buffered I/O) như BufferedReader và BufferedWriter.

1. Đọc File (BufferedReader)

Sử dụng BufferedReader kết hợp với FileReader giúp tăng hiệu suất đọc từng dòng một cách dễ dàng và an toàn (sử dụng khối try-with-resources để tự động đóng luồng).


import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class FileReadExample {
    public static void main(String[] args) {
        String fileName = "config.txt";
        try (BufferedReader reader = new BufferedReader(new FileReader(fileName))) {
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println("Đọc: " + line);
            }
        } catch (IOException e) {
            System.err.println("Lỗi đọc file: " + e.getMessage());
        }
    }
}

2. Ghi File (BufferedWriter)

Tương tự, BufferedWriter và FileWriter được dùng để ghi dữ liệu. Tham số thứ hai của FileWriter (true) cho phép bạn ghi nối tiếp (append) vào cuối file thay vì ghi đè.


import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class FileWriteExample {
    public static void main(String[] args) {
        String fileName = "log.txt";
        String content = "Ghi log lúc: " + new java.util.Date();
        try (BufferedWriter writer = new BufferedWriter(new FileWriter(fileName, true))) {
            writer.write(content);
            writer.newLine(); // Thêm xuống dòng
            System.out.println("Đã ghi log thành công.");
        } catch (IOException e) {
            System.err.println("Lỗi ghi file: " + e.getMessage());
        }
    }
}
