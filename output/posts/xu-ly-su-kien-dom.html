Xử lý sự kiện DOM trong JavaScript

   

JavaScript Frontend

   

DOM (Document Object Model) là một giao diện lập trình cho các tài liệu HTML. Xử lý sự kiện DOM là việc sử dụng JavaScript để lắng nghe các tương tác của người dùng (như click chuột, gõ phím, di chuột) và thực hiện các hành động phản hồi.

Phương thức chuẩn: addEventListener()

addEventListener() là cách thức linh hoạt nhất để gán nhiều hàm xử lý cho cùng một sự kiện và dễ dàng loại bỏ chúng.


// 1. Lấy phần tử (Element)
const button = document.getElementById('domButton');
const message = document.getElementById('domMessage');

// 2. Định nghĩa hàm xử lý (Handler)
function handleButtonClick() {
    message.textContent = "Nút đã được bấm!";
    button.classList.add('btn-success');
    button.classList.remove('btn-primary');
}

// 3. Gán sự kiện 'click'
button.addEventListener('click', handleButtonClick);

// Ví dụ sự kiện 'mouseover'
button.addEventListener('mouseover', () => {
    message.textContent = "Bạn đang di chuột qua nút.";
});

// Có thể dùng removeEventListener('click', handleButtonClick); để gỡ bỏ.

Các sự kiện phổ biến khác bao gồm: submit (trên form), keydown (gõ phím), change (thay đổi giá trị input), và load (tải trang).
